## Getting Started
### Prerequisites
* [Node.js](https://nodejs.org/en/)
* [npm](https://npmjs.com/getnpm)
* [MySQL](#)

### Installation
1. Clone the repo
```sh
git clone https://github.com/mpieces/bamazon.git
```
2. Install NPM packages in root directory
```sh
npm install
```

## Usage
1. `node bamazon.js`
